# RetailGenius – AI Customer Churn Prediction

This project implements an AI-based customer churn prediction system as part of the EPITA
AI Project Methodology course (Part 2 – Technical Methodology).

## Project Goals
- Build a reproducible ML pipeline for churn prediction
- Apply software engineering best practices
- Track experiments and models using MLflow

## Project Structure
This project follows a cookiecutter-style data science structure:
- data/: raw and processed datasets
- notebooks/: exploratory analysis
- src/: data preparation, feature engineering, modeling
- mlruns/: MLflow experiment tracking

## Tools
- Python
- Scikit-learn
- MLflow
- Git & GitHub
